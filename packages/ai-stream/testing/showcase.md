# Промпт: демонстрация всех возможностей markdown

Сгенерируй один цельный markdown-документ, который компактно демонстрирует максимум возможностей разметки.

Обязательно включи:
- заголовки `#` и `##`;
- обычный текст, **жирный**, _курсив_, ~~зачеркнутый~~, `inline code`;
- цитату `>` и вложенную цитату;
- маркированный, нумерованный и вложенный списки;
- список задач (`- [ ]`, `- [x]`);
- таблицу минимум 4x4;
- горизонтальную линию `---`;
- блок кода с указанием языка;
- ссылку и изображение (можно с фиктивным URL);
- короткий блок `HTML` внутри markdown;
- сноску (`[^1]`) и определение сноски в конце;
- секцию с экранированием спецсимволов markdown.

Требования:
- документ должен быть читаемым и логичным, не просто набором фрагментов;
- без служебных комментариев про форматирование;
- не используй заголовки глубже `##`.
