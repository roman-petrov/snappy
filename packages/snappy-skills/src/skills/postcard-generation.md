# Skill: Postcard Generation

## Goal

Produce a complete postcard visual concept with aligned message, composition, and mood.

## Use When

- User asks for greeting card/postcard artwork for holiday, event, or personal occasion.
- User needs printable or digital card visuals with optional embedded text.

## Ready For Generation Criteria

- Occasion and recipient style are identified.
- Card format and orientation are selected.
- Visual direction and message placement are defined.
- Constraints for text readability and safe margins are clear.

## User Discovery Topics

- Occasion, emotional tone, and cultural context.
- Recipient profile and preferred aesthetics.
- Visual style and motif palette (illustrative, minimal, festive, elegant, retro).
- Color direction and seasonal associations.
- Message handling (no text, short line, headline + subtitle).
- Composition requirements (foreground subject, background density, whitespace).
- Output specifics (print vs digital, ratio, bleed-safe expectations).

## Clarification Strategy

1. Minimum baseline: determine occasion, style, orientation, and whether text is embedded.
2. Design alignment: clarify motif hierarchy, palette, typography intent, and layout balance.
3. Final validation: confirm production context and final acceptance criteria.

## Quality Checklist

- Composition supports quick emotional read.
- Text area remains legible and uncluttered.
- Palette and style fit the occasion.
- Message and imagery do not conflict in tone.
- Prompt includes negative constraints to prevent over-detail.

## Common Failure Modes And Prevention

- Crowded layout: reserve breathing zones and define maximum element density.
- Weak occasion signaling: require explicit symbolic anchors tied to event context.
- Poor text legibility: specify contrast, safe area, and background simplicity behind text.
