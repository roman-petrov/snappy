# 🔥 KISS, YAGNI, Occam

- **id:** `programming/kiss-yagni-occam`
- **emoji:** 🔥
- **applies:** `*`

## 📐 Norm

Keep it simple and compact. No code for the future — only what is needed now. Delete anything unnecessary (Occam's
razor). Prefer maximally concise code.

## 🔍 Detect

Look for speculative abstractions, unused flexibility, dead branches added "for later".

## 🔧 Fix

Remove speculative code; keep the smallest solution.
