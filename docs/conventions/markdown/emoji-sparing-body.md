# 😀 Emoji sparingly in body

- **id:** `markdown/emoji-sparing-body`
- **emoji:** 😀
- **applies:** `**/*.{md,mdc}`

## 📐 Norm

Use emoji sparingly in body text.

## 🔍 Detect

Body paragraphs crowded with decorative emoji.

## 🔧 Fix

Remove decorative body emoji; keep only when they add clear meaning.

## 📝 Examples

### ❌ Bad

`🚀 Ship it 🎉 today ✨ after review ✅`

### ✅ Good

`Ship it today after review.`
