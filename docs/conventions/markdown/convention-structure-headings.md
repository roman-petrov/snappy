# 😀 Convention structure headings

- **id:** `markdown/convention-structure-headings`
- **emoji:** 😀
- **applies:** `docs/conventions/**/*.md`

## 📐 Norm

Structure headings always use the fixed set from `docs/conventions/README.md`:

- `📐 Norm`
- `🔍 Detect`
- `🔧 Fix`
- `📝 Examples`
- `❌ Bad`
- `✅ Good`

## 🔍 Detect

Convention atom structure headings that diverge from the fixed set.

## 🔧 Fix

Restore atom structure headings to the fixed set.

## 📝 Examples

### ❌ Bad

`## Norm`

### ✅ Good

`## 📐 Norm`
