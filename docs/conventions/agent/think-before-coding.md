# 🤔 Think before coding

- **id:** `agent/think-before-coding`
- **emoji:** 🤔
- **applies:** `*`

## 📐 Norm

- Don't assume.
- Don't hide confusion.
- Surface tradeoffs.
- State assumptions.
- If uncertain or unclear, stop and ask.
- If multiple interpretations exist, present them.
- If a simpler approach exists, say so.

## 🔍 Detect

N/A (behavioral).

## 🔧 Fix

Ask or present options before implementing when assumptions are unclear.
