# 🌐 Website design

## 💡 Concept

**Agent skill.** Shapes a landing or site from goals and audience: what sections exist, in what order, what each block
should say and show. Can offer several structure or mood variants before anyone opens a design tool.

## 👤 How the user experiences it

1. The user states the site goal (sell, explain, collect leads), audience, and what content they already have.
2. The agent proposes **page structure**—sections and purpose of each.
3. After approval, the agent fills in headlines, body outlines, image ideas, and calls to action.
4. The user gets a wireframe-level brief or a stronger visual direction to hand to implementation; they may request a
   second variant to compare.
