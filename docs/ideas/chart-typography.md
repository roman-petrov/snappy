# 🔤 Chart typography

## 💡 Concept

**Extension of chart generation.** When building a chart, the user chooses fonts for title, axes, and legend—from Snappy
defaults or their own file—so the result matches brand typography.

## 👤 How the user experiences it

1. While creating a chart, the user opens font settings.
2. They pick one font for everything or separate fonts for title and labels.
3. If they upload a font, they confirm they may use it.
4. The exported chart carries the chosen typography; Snappy notes fallbacks if some symbols are missing.
