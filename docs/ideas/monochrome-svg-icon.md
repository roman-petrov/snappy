# ⬛ Monochrome SVG icon

## 💡 Concept

**Agent skill.** Delivers a one-color vector icon: clean shapes, no photo inside, readable even at very small sizes—for
toolbars, favicons, and brand rules that allow only a single ink color.

## 👤 How the user experiences it

1. The user describes what the icon should mean and where it will appear.
2. The agent clarifies style (rounded, sharp, outline vs filled).
3. The user sees previews at small sizes on light and dark backgrounds.
4. They download **SVG** and, if needed, ask for small adjustments to weight or silhouette.
