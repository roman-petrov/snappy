# 🎨 Color SVG icon

## 💡 Concept

**Agent skill.** A flat or limited-color vector icon with a clear role for each color, aligned with the product palette
when the user wants consistency across the app.

## 👤 How the user experiences it

1. The user describes the metaphor and style (flat, duotone, soft gradient accent).
2. They choose a small set of colors or tie the icon to Snappy theme colors.
3. The agent shows the icon at several sizes and on light and dark UI backgrounds.
4. The user downloads editable **SVG** for design tools or the app.
