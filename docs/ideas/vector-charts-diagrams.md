# 📊 Vector charts & diagrams

## 💡 Concept

**Agent skill.** Builds charts and simple diagrams from the user’s numbers or lists—bars, lines, pie, basic flows—so
values on the graphic match what the user entered, as editable vector output rather than a fuzzy AI screenshot.

## 👤 How the user experiences it

1. The user pastes a table or enters data in a simple form.
2. They choose chart or diagram type and a visual style.
3. The agent may suggest a better chart type if the data fits another view.
4. The user previews the graphic, fixes labels if needed, and downloads **SVG** (and optionally a PNG preview for
   slides).
