# 📄 Micro-course generator

## 💡 Concept

**Agent skill.** Turns a topic and audience level into a short, structured lesson: modules, key points, examples, and a
recap. The user adjusts scope and depth before the agent writes content.

## 👤 How the user experiences it

1. The user names a topic, who it is for (beginner, intermediate, expert), and how much time they have.
2. The agent proposes a **module outline** with rough timing; the user can drop, add, or deepen sections.
3. The agent fills each module with explanations and examples, streamed in the chat.
4. The user reads in Snappy or downloads a **PDF**, or opens an **interactive** version with sections and optional
   quick checks at the end.
