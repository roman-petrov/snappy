# ✏️ Prompt image editing

## 💡 Concept

**Agent skill + image tool.** The user uploads a picture and describes what to change: remove an object, replace a
detail, repaint a region, or restyle the whole image while keeping the overall composition when possible.

## 👤 How the user experiences it

1. The user uploads an image.
2. They describe the edit in words; optionally mark an area on the image to change only that part.
3. The agent confirms what will change and what will stay the same.
4. A new version appears; the user compares before and after and can iterate or download the result.
