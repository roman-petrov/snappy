# 🔲 Branded QR codes

## 💡 Concept

**Agent skill + generator.** Makes a QR code that still scans reliably after adding brand colors, rounded dots, and an
optional logo in the center.

## 👤 How the user experiences it

1. The user enters a link or other payload (contact card, Wi‑Fi, and so on).
2. They pick colors and style; optionally add a small logo.
3. Snappy checks that the code still **scans** in preview.
4. The user downloads **SVG** for print or **PNG** for screens and documents.
