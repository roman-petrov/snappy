# ✒️ Vector SVG logo

## 💡 Concept

**Agent skill.** Explores logo directions, then delivers real vector artwork: symbol only, text only, or combined—with
variants for different layouts, not a flat picture pretending to be a logo.

## 👤 How the user experiences it

1. The user shares brand name, field, and the feeling the logo should convey.
2. The agent offers several metaphor directions; the user picks one.
3. The agent presents lockups (horizontal, stacked, icon-only) as **SVG**.
4. The user receives simple guidance on clear space and minimum size for real-world use.
