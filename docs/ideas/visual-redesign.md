# 🔄 Visual redesign

## 💡 Concept

**Agent skill.** Starts from a screenshot, mockup, or photo plus the user’s goals (modern look, new colors, simpler
layout). The user marks what must not move—logo, headline, menu—and the agent suggests new directions.

## 👤 How the user experiences it

1. The user uploads a reference image and says what should improve.
2. They list elements that must stay (logo place, main text, structure).
3. The agent shows several **visual directions** as images or clear layout descriptions.
4. The user picks one and asks for refinements, then uses the result as a brief for design or development.
