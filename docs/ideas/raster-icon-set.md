# 📦 Raster icon set

## 💡 Concept

**Export step** after an icon is ready (often from the mono or color SVG flow). One approved icon becomes a zip of PNG or
WebP files in standard sizes, consistently padded and centered.

## 👤 How the user experiences it

1. The user has an SVG they like (from Snappy or upload).
2. They tick the sizes they need (16 through 512 and common mobile scales).
3. Snappy generates all files in one **zip** with clear names.
4. The user drops the set into their app or store listing without manual resizing.
