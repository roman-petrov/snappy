# 🖼️ Wallpaper generator

## 💡 Concept

**Agent skill.** Creates backgrounds sized for the user’s device (phone, tablet, desktop, extra-wide screens), with
control over mood, colors, and how busy the image is. The UI can show where icons and widgets will sit so the wallpaper
stays readable.

## 👤 How the user experiences it

1. The user picks a device type or screen shape.
2. They choose mood, palette, and whether the image should stay calm behind icons or carry a strong subject.
3. The agent offers one or several variants; the user previews them with a **safe-area overlay** on phone layouts.
4. The user downloads the wallpaper, sometimes with light and dark variants or a tighter crop.
