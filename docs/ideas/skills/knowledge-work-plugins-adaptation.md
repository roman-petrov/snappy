# 🔍 Knowledge-work plugins study

**Reference:** [anthropics/knowledge-work-plugins](https://github.com/anthropics/knowledge-work-plugins)

## 💡 Concept

**Exploration, not a Cowork integration.** Snappy reviews Anthropic’s open plugin collection for Claude Cowork: how
roles are defined, what each plugin’s skills and commands teach the agent, and which workflows look strong for knowledge
workers. The goal is to learn from a mature public reference and decide whether—and how—anything should be adapted for
Snappy. The outcome is open: new agent skills, presets, UX patterns, a subset of workflows, or no port at all after the
review.

For the product team and anyone shaping Snappy’s agent catalog—not for users who expect to run Cowork plugins inside
Snappy.
