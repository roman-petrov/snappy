# 🧪 Pixel-shader micro-agent

## 💡 Concept

**Agent skill (Lab).** A creative agent that writes and refines visual effects as shader code: abstract scenes, 3D-style
text, festive postcards, or charts driven by user data. Live preview on canvas; optional short video export. Aimed at
experimentation, not everyday documents.

## 👤 How the user experiences it

1. The user describes the effect they want (mood, colors, motion, optional data file).
2. The agent generates code; the **canvas preview** updates when it compiles.
3. The user asks for changes in plain language (“slower”, “more glow”, “different palette”).
4. When happy, they record a loop or export a short clip to share.
