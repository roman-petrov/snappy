# 🖥️ Presentation builder

## 💡 Concept

**Agent skill.** Builds a slide deck from a short brief: first an outline, then slides with titles, bullet points,
speaker notes, and hints for visuals. For pitches, lessons, and team updates without starting from a blank deck.

## 👤 How the user experiences it

1. The user describes the goal, audience, and how long the talk should be.
2. The agent shows a **slide outline** (titles and purpose of each slide); the user reorders or edits it.
3. The agent fills the deck: slide text, notes for the speaker, and what kind of image or chart would fit.
4. The user previews slides in Snappy or exports **PDF** or **PowerPoint**, or presents from an in-app player.
